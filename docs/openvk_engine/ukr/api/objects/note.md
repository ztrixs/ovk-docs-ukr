# Об'єкт ноталкі

| Властивість | Значення | Опис |
|----------|----------|----------|
| id | int | ID ноталкі |
| owner_id | int | ID власника ноталкі |
| title | string | Назва ноталкі |
| text | string | Зміст нотатки |
| date | int | Час створення нотатки у форматі UNIX |
| comments | int | Кількість коментарів під ноталкою |
